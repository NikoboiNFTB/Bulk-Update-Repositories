### Turbo

`clone-author` with parallel processing.

Default (32 parallel clone operations)

```bash
bash clone-author-turbo NikoboiNFTB
```

Laptop on battery? Chill mode

```bash
JOBS=2 bash clone-author-turbo NikoboiNFTB
```

Change JOBS value to change amount of background jobs.
