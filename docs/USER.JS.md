## Auto-Reload if Pending Deployment

Automatically reloads your GitHub Pages repository if it's pending deployment, allowing you to automatically see when it's done.

**Installation**

Click [here](https://github.com/NikoboiNFTB/GitHub-Tools/raw/refs/heads/main/userscript/auto-reload.user.js).

## Deletion Process Auto-Fillers

These scripts automatically fill in the GitHub repository deletion steps.

### Full-Auto

Automatically fills in all repo deletion steps. Because there are too many.

**Installation**

Clicke [here](https://github.com/NikoboiNFTB/GitHub-Tools/raw/refs/heads/main/userscript/delete-full-auto.user.js).


### Semi-Auto

Automatically fills in the text repo deletion step.

**Installation**

Click [here](https://github.com/NikoboiNFTB/GitHub-Tools/raw/refs/heads/main/userscript/delete-full-auto.user.js).
